Add your images here:

- `university-cincinnati-logo.png` - University logo
- `bellevue-college-logo.png` - Bellevue College logo
- `background-hero.jpg` - Hero section background image (if desired)

## Image Guidelines

### Profile Photo
- Professional headshot with good lighting
- Plain or professional background
- Business casual or professional attire
- Face clearly visible
- Square format (400x400px to 800x800px)
- PNG or JPG format

### Company Logos
- Official company logos (check usage rights)
- Transparent background (PNG format preferred)
- Recommended size: 200x200px or similar
- High resolution for retina displays

### Optimization
Before adding images, optimize them:
- Use tools like TinyPNG, ImageOptim, or Squoosh
- Keep file sizes under 200KB for photos
- Keep logos under 50KB
- Use modern formats (WebP) if possible

## Where Images Are Used

- **Hero Section**: Profile photo (optional)
- **About Section**: Profile photo
- **Experience Section**: Company logos
- **Education Section**: University logos (optional)

## How to Add Images

1. Save your images in this folder (`src/assets/images/`)
2. Images are automatically imported in components
3. Update the image filenames in components if needed
4. Images will be optimized by Vite during build

## Placeholder System

If an image is missing:
- Components will show a fallback gradient or icon
- No broken images will appear
- Portfolio remains professional even without images
